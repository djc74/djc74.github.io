Projects

Uplift

For my React project at General Assembly, I wanted to make a site that gathered together reasons to be cheerful. Uplift displays jokes, quotes, gifs and pictures pulled in from APIs and allows logged in users to save ones they particularly like to their own page. They can also upload their own examples, with uploaded files stored on Amazon web services.

Gig.ly

For our four-person group project at General Assembly, we built an app for people to plan trips to concerts together. We used the Ticketmaster API to provide a list of upcoming events in London, linked to Google Maps to show where the venue was, and the Spotify API to allow tracks from the artist to be played - using OAuth for signing in - as well as a chat forum.

Monstergram

After I made a JavaScript game for my four-year-old son, my other son, who’s seven, wanted me to make something for him. He likes drawing pictures of dragons and monsters he’s made up, so for this project I created an app that would allow him to login, save these pictures and comment on them. Using MongoDB, Express, AngularJS and Node.js, I built a full-stack app that he’s rapidly filling with monstrous creations.

Feed The Monster!

When I asked my four-year-old son what game he would like me to make him, he said one where you have to feed sausages to a space monster to make him explode before he eats you. So using JQuery, HTML and CSS, I made a version of whack-a-mole with a space/sausages theme.
